## 2. MANDATORY POST-TASK WORKFLOW (VERY IMPORTANT)

Immediately after completing any task (creating a file, writing logic, fixing a bug, refactoring code, etc.), you **MUST** document your progress by appending a log entry to:
👉 [task-log.md](file:///ai-agent/task-log.md)

### Required Entry Format:
Each entry should be in Vietnamese and include:
1. **Tiêu đề**: Tên task và ngày thực hiện.
2. **Công việc đã thực hiện**: Liệt kê chi tiết các files đã tạo/sửa đổi và logic đã viết.
3. **Công nghệ áp dụng**: Các API của Roblox, ModuleScript, OOP, Rojo, Git, v.v.
4. **Kiến thức đã áp dụng**: Các giải pháp kỹ thuật, mẫu thiết kế hoặc thuật toán cụ thể (ví dụ: Weighted Random, DataStore Caching, AlignPosition Physics, v.v.).

Ensure this rule is strictly followed to provide the user with a clear, high-level overview of the project's evolution.
